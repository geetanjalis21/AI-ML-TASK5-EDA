# AL-ML-TASK5: Decision Trees & Random Forests

## Objective
To understand and implement tree-based models for classification.

## Tools Used
- Python
- Scikit-learn
- Matplotlib

## Steps Performed
- Loaded and explored dataset
- Trained Decision Tree Classifier
- Controlled overfitting using max_depth
- Trained Random Forest model
- Compared accuracy
- Analyzed feature importance
- Performed cross-validation
- Visualized decision tree

## Results
- Random Forest performed better than Decision Tree
- Feature importance highlighted key factors affecting prediction

## Conclusion
Tree-based models are powerful and Random Forest improves accuracy by reducing overfitting.
